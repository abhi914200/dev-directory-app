# Developer Directory App

Simple Full-Stack app to add and list developer profiles. Built as a learning project.

## What it does
- Add developer: name, role, tech stack, experience
- List developers with search & filter (by role or tech)
- Stores data in a local JSON file (backend/data.json)
- Clean UI with Tailwind, React (Vite), and Express (Node)

## Tech
- Frontend: React (Vite), Tailwind CSS
- Backend: Node.js, Express (ES Modules)
- Storage: local JSON file (low-friction for beginners)
- Dev tooling: nodemon (backend), Vite (frontend)

## Local setup

### Prerequisites
- Node.js (LTS recommended)
- Git

### Start backend
```bash
cd backend
npm install
npm run dev
# backend runs at http://localhost:4000
